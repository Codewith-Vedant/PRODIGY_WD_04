Tic-Tac-Toe

The Tic Tac Toe web application is an engaging and interactive game designed for two players. It features a clean and intuitive interface built with HTML, CSS, and JavaScript. The game board consists of a 3x3 grid, and players take turns marking cells with their respective symbols ("X" or "O"). The dynamic interface updates in real-time, providing a responsive and enjoyable user experience.Players can enjoy a classic game of Tic Tac Toe, testing their strategic skills against a friend or the computer/AI.
